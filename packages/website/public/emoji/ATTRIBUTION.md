# Fluent Emoji

The SVG files in this directory are taken from Microsoft's
[Fluent Emoji](https://github.com/microsoft/fluentui-emoji) collection
(the "Color" set, 3D style).

License: MIT — see https://github.com/microsoft/fluentui-emoji/blob/main/LICENSE

Used for the "Why AnyUI" feature cards on the landing page. Files are
copied verbatim (no modifications) from the upstream `assets/<Name>/Color/`
folders.

| File        | Upstream emoji             |
| ----------- | -------------------------- |
| cupcake.svg | Cupcake                    |
| trident.svg | Trident emblem             |
| ice.svg     | Ice                        |
| palette.svg | Artist palette             |
| wand.svg    | Magic wand                 |
| package.svg | Package                    |
